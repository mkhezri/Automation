# KAU AI Agent Platform

Open-source AI Agent platform for Mining and Commodity Trading Companies.

## Overview
This project aims to automate commercial and procurement workflows in mining and international trading organizations.

## Features
- Tender evaluation
- Contract review
- Supplier assessment
- Shipment monitoring
- Document generation
- Business reporting
- Email automation
